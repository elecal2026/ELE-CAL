# ELE-CAL 消費税・請求まわりの整理

> 作成日: 2025-05-05  
> ステータス: 免税事業者（消費税の納税義務なし）

---

## 現在の状況

| 項目 | 内容 |
|---|---|
| 事業者区分 | **免税事業者**（年間売上1,000万円以下） |
| インボイス登録 | **未登録** |
| Stripe 商品税コード | `txcd_10103001`（SaaS - 業務使用）✅ 正しい |
| Stripe 税務登録 | **未登録**（日本での税金徴収を登録していない） |
| 請求金額 | ¥500／月（消費税なし） |

---

## 免税事業者のルール（かんたんまとめ）

### ✅ できること
- 消費税を**含んだ価格**で販売すること（例：¥550を「税込価格」として設定）
- 普通の請求書を発行すること
- 消費税を**納めなくてよい**（免除されている）

### ❌ できないこと
- 消費税を**別途上乗せして請求**すること（例：¥500 + 消費税¥50 = ¥550 と分けて表示）
- **適格請求書（インボイス）** を発行すること
- お客さん側が仕入税額控除（経費として消費税を差し引くこと）に使える書類を出すこと

---

## 今のStripe設定は正しいのか？

**→ はい、正しい。**

- 消費税が「-」と表示されている ＝ 消費税を請求していない
- 免税事業者なので、この動作で正しい
- Stripeの税務登録（日本）をしていない ＝ 消費税を計算しない ＝ 正しい挙動

---

## 将来やるべきこと（売上が増えたとき）

### いつ？
- **年間売上が1,000万円を超えた翌々年**から消費税の納税義務が発生
- または、自分から**インボイス登録**をした場合

### やること
1. 税務署に「消費税課税事業者届出書」を提出
2. インボイス発行事業者の登録申請
3. Stripeダッシュボードで **税務登録を追加**（設定 → Tax → 日本を登録）
4. 価格を ¥500 + 消費税¥50 = ¥550 に変更（または税込¥550に設定）
5. 消費税の確定申告（年1回）

### Stripeでの具体的な手順
1. **設定 → Tax** を開く
2. **「税務登録を追加する」** をクリック
3. 場所：日本 → **「すでに登録済みです」** を選択
4. 登録番号（T + 13桁の番号）を入力
5. 保存
