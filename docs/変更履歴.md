# 変更履歴

> 機能変更を **追記式** で記録する。古いエントリは消さない。
> 1エントリ = 1日付（または1リリース）。新しいものを上に。
>
> フォーマット：
> ```
> ## YYYY-MM-DD — 概要
> - **変更**: 何を変えたか
> - **理由**: なぜ変えたか（または 設計判断.md#xxx 参照）
> - **影響範囲**: 触ったファイル / 画面
> - **関連**: コミット / Issue / やること一覧.md の項目
> ```
