## 3. ブレーカー選定ツール 仕様

### 3.1 画面構成

`src/app/breaker/page.tsx` を中心に、`LoadEntryRow.tsx` / `ResultPanel.tsx` / `validation.ts` で構成。

レイアウトは左右2カラム：
- **左カラム**：基本設定 + 負荷リスト入力
- **右カラム**：計算結果 + 注意事項 + 検討事項

### 3.2 基本設定（設定バー）

| 項目 | 選択肢 | デフォルト | 備考 |
|---|---|---|---|
| 配電方式 | 単相2線 / 単相3線 / 三相3線 | 三相3線 | |
| 定格電圧 | 100V / 200V | 200V | 三相3線時は100Vを自動で disabled、選択中なら200Vへ自動切替 |
| 力率 cosθ | 1.0 / 0.9 / 0.8 | 0.8 | |
| 余裕率 | 1.0倍 / 1.25倍 / 1.5倍 | 1.25倍 | ブレーカー選定マージン |

### 3.3 負荷リスト

各負荷エントリ（`LoadEntry` 型）の項目：

| フィールド | 型 | 内容 |
|---|---|---|
| `id` | string | 内部ID |
| `name` | string | 負荷名称（自由入力） |
| `type` | `'general' \| 'motor' \| 'welder'` 等 | 負荷種別。**現状は1負荷ごとに自由選択（混在可）** |
| `powerKw` | string | 電力 [kW] |
| `startMethod` | `'direct'` 等 | モーター始動方式 |
| `usageRate` | string | 溶接機の使用率 [%]。デフォルト50 |
| `wiring` | `{ wireType, wireSize, wireLength }` | 電線情報（**現状は1負荷=1電線**） |

操作：
- 「+ 負荷を追加」で無制限に追加可能
- 各行の削除ボタン（最後の1行は削除不可）

### 3.4 計算ロジック（現状）

- **合計kW計算**：各負荷の `powerKw` を合算。溶接機は `kw × √(usageRate/100)` で換算（力率0.6想定の固定換算）
- **電流算出**：`I = P / (K × V × cosθ)`、K = 1（単相）/ √3（三相）
- **ブレーカー選定**：内線規程テーブルを引く。詳細は `ResultPanel.tsx` 参照
- **バリデーション**：`validation.ts` の `validateBreakerInputs` が `{ target: 行index, ... }` 形式で警告を返す

### 3.5 結果表示

- 右カラム上部に `ResultPanel`（選定結果・電流値など）
- 警告・エラーは現状 `ResultPanel` 内に表示（**画面下方になりがちで気付きにくい問題あり** → 改善対象）

### 3.6 既知の検討事項（コード内 `<details>` に記載）

- **B-14**: モーター・溶接機が混在する場合の優先順位（現状モーター優先）
- **B-15**: 複数台モーターの幹線テーブル（3705-3表）引き機能
- **B-16**: 溶接機の力率0.6固定の妥当性

## 4. データモデル（現状の前提）

```
ブレーカー（1台）
  └─ 負荷[] （複数）
       └─ 電線（1本）  ← ここが「1負荷=1電線」前提
```

> この前提を **「1負荷 → 1配管 → 電線[複数]」** に拡張する話が
> やること一覧.md のフェーズ3にある。実施時はここの構造図を更新すること。
