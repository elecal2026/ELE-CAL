## 2. ケーブルラック敷設の施工基準（出典①）

### 2.1 ケーブルラックの取付け・支持 — 2.10.1（PDF p76）

以下は公共建築工事標準仕様書の原文より抜粋。

**水平支持間隔**（2.10.1(2)）
> ケーブルラックの水平支持間隔は、鋼製では**2ｍ以下**、その他については**1.5ｍ以下**とする。

**垂直支持間隔**（2.10.1(3)）
> ケーブルラックの垂直支持間隔は、**3ｍ以下**とする。ただし、配線室等の部分は、6ｍ以下の範囲で各階支持とすることができる。

**つりボルト径**（2.10.1(4)）
> ケーブルラックを支持するつりボルトは、ケーブルラックの幅が**呼び600mm以下のものでは呼び径9mm以上**、**呼び600mmを超えるものでは呼び径12mm以上**とする。

**終端部**（2.10.1(5)）
> 終端部には、**エンドカバー又は端末保護キャップ**を設ける。

**アルミ製ラックの異種金属腐食**（2.10.1(6)）
> アルミ製ケーブルラックは、支持物との間に**異種金属接触腐食を起こさないように**取付ける。

**本体相互の接続**（2.10.1(7)）
> ケーブルラック本体相互間は、ボルト等により**機械的、かつ、電気的に接続**する。

**自在継手・エキスパンション部のボンディング**（2.10.1(8)）
> ケーブルラックの自在継手部及びエキスパンション部には、**ボンディングを施し、電気的に接続**する。

**屋外カバーの固定**（2.10.1(10)）
> 屋外に設けるケーブルラックにカバーを取付ける場合は、カバーが**飛散しないように止め金具、バンド等で確実に取付ける**。

**MD化時の補足**:
- 2.10.1(1) には、ケーブルラック又は支持金物をスラブ等の構造体に取り付ける旨の施工条件がある。現行UIの計算値には未使用。
- 2.10.1(9) には、ボンディング用接続線は 2.2.5「管の接続」(6) による旨の参照がある。現行UIの注記では詳細仕様までは表示していない。

### 2.2 ケーブルの曲げ半径 — 2.10.4.1 共通事項（PDF p77）

**表2.10.1 ケーブルの曲げ半径**（曲げ半径は内側半径）

| ケーブルの種別 | 単心以外 | 単心 |
|---|---|---|
| 低圧ケーブル | 仕上り外径の**6倍以上** | 仕上り外径の**8倍以上** |
| 低圧遮へい付ケーブル / 高圧ケーブル | 仕上り外径の**8倍以上** | 仕上り外径の**10倍以上** |

> 備考: デュプレックス形、トリプレックス形（CVT）及びカドラプレックス形のより線における仕上り外径は、より合せ外径をいい、「単心以外」を適用する。

**ELE-CALへの適用**:
- CV 1C（単心）→ ×8
- CV 2C/3C/4C（多心）→ ×6
- CVT（トリプレックス）→ ×6（より合わせ外径に「単心以外」を適用）
- VVF → ×6

### 2.3 ケーブルラック配線のケーブル固定 — 2.10.4.3（PDF p78）

**固定間隔**（2.10.4.3(1)(ｱ)）
> ケーブルは、整然と並べ、**水平部では3ｍ以下、垂直部では1.5ｍ以下**の間隔ごとに固定する。ただし、トレー形ケーブルラック水平部の場合は、この限りでない。

**垂直敷設時の荷重**（2.10.4.3(1)(ｲ)）
> ケーブルを垂直に敷設する場合は、**特定の子げたに荷重が集中しないように固定**する。

**電力ケーブルの積重ね禁止**（2.10.4.3(1)(ｳ)）
> 電力ケーブルは、積重ねを行ってはならない。ただし、次のいずれかの場合は、この限りでない。
> - (a) 単心ケーブルの俵積み
> - (b) 分電盤2次側のケーブル
> - (c) 積重ねるケーブルの許容電流について必要な補正を行い、配線の太さに影響がない場合

**→ ELE-CALでの扱い**: 「電力ケーブルは原則として1段並べで選定」として注意表示。

## 4. ラック幅計算式（ELE-CAL独自）

### 4.1 計算根拠の分類

| 項目 | 根拠 | 備考 |
|---|---|---|
| ケーブル占有幅 = Σ(外径×本数) | 数学的定義 | 根拠資料なし |
| 並べ方の余裕係数（1.00〜1.30） | **ELE-CAL独自** | 公共建築工事標準仕様書に規定なし。施工余裕・蛇行・結束を見込む簡易係数 |
| 将来増設率（0〜50%） | **ELE-CAL独自** | 同上 |
| ラック幅候補: 200, 300, 400, 500, 600, 800, 1000mm | 市場標準品の幅ラインナップ | 規格表に基づく代表的な製品寸法 |

### 4.2 必要幅の計算式

```
ケーブル占有幅 = Σ(ケーブル外径 × 本数)
必要幅 = ケーブル占有幅 × 並べ方の余裕 × (1 + 将来増設率)
推奨ラック幅 = 幅候補のうち必要幅以上の最小幅
```

## 5. 内線規程のラック工事規定（参考）

### 5.1 ラックへの施設（3165-2(5)）

> ラックなどに施設する場合は、次の各号によること。
> ①ラックなどは、**ケーブルの重量に十分耐える構造**であって、かつ、**堅固に施設**すること。
> ②ラックなどにケーブルを施設する場合のケーブルの支持点間の距離は、**ケーブルが移動しない距離**とすること。

### 5.2 接地（3165-8）

> ケーブルの金属被覆ケーブルの金属製附属品、金属製の電線接続箱など、管その他の防護装置の金属製部分及びラックなどの金属製部分は、3165-8（接地）に準じて接地工事を施すこと。

→ 用途: 接地・施工注意事項の注記として参照。

### 5.3 ラック幅・積載荷重の選定規定

**内線規程 JEAC8001-2022 には「ラック幅の選定計算表」は存在しない。**

配管選定の「内断面積の32%表」（3110-9〜11表）に相当する規定がなく、公共建築工事標準仕様書も同様にラック幅算定式を定めていない。

## 7. PDF目視照合結果と実装修正候補

### 7.1 PDF目視で確認できたこと

- 公共建築標準仕様書 PDF p76-p78 は `/cable_rack` の施工条件（支持間隔、つりボルト径、曲げ半径、固定間隔、積重ね禁止）の直接根拠として使える。
- 公共建築設備工事標準図は、現行 `/cable_rack` の計算値・表示条件では直接参照していない。
- 内線規程 PDF p856-p859 は CE/F の仕上り外径表として使えるが、質量は掲載されていない。
- 内線規程 PDF p835 は OW/DV の構造表であり、現行 `/cable_rack` の CV/CVT/VVF 外径表の直接根拠ではない。

### 7.2 現行実装との差分

- `src/app/cable_rack/page.tsx` の `CABLE_TABLE` の `od` は、内線規程 PDF p856-p859 の CE/F 仕上り外径と大きく異なる。PDF原本を根拠にするなら、外径値は 3.2 の安全側採用候補へ差し替え候補。
- `CABLE_TABLE` の `mass` は、今回のPDF画像では根拠確認できない。メーカー資料を別途根拠化するか、UI上で「代表値」として扱う必要がある。
- `CVT` と `VVF` の外径は、今回のPDF画像では原本確認できない。メーカー資料又は内線規程の該当ページを追加確認する必要がある。
- 実装のボンディング注意は `hasCover` が真の場合だけ表示されるが、公共建築標準仕様書 2.10.1(7)(8) のボンディング条件はカバー有無に限定されない。表示条件の見直し候補。
- `ラック幅候補`、`並べ方の余裕係数`、`将来増設率`、`ラック幅計算式` は、公共建築標準仕様書・内線規程の規定値ではなく ELE-CAL 独自の簡易選定条件として扱う。
