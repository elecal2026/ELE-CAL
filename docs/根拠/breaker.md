## 5. 需要率（3605-11表）

参照ページ: **PDF p779**（紙面p697）

10kVA を超える容量に対する需要率：

| 建物の種類 | 需要率 |
|---|---|
| 住宅・寮・アパート・旅館・ホテル・病院・倉庫・寺院・教会 | 50% |
| 学校・事務所・銀行 | 70% |

**備考**: 上記の建物に1台ごとの定格容量が10kVAを超える大形電気機器がある場合は、
```
需要率 = (10kVA × 50%) + (大形電気機器の超過分kVA × 100%)
```
で算出する。

## 7. 幹線の過電流遮断器選定（電技解釈 第148条）

### 7.1 法的上限の式

```
電動機負荷がある場合:
  I_B ≤ 3 × I_M + I_H
  (I_M > 50A の場合: I_B ≤ 2.75 × I_M + I_H)
  I_M: 電動機定格電流の合計
  I_H: その他の負荷電流の合計

電線の許容電流に基づく上限:
  I_B ≤ 2.5 × I_W
  I_W: 幹線の許容電流

上記2式のうち「小さい方」を上限とする
```

### 7.2 実務上の選定マージン

| 負荷種別 | 推奨選定係数 | 理由 |
|---|---|---|
| 一般電灯・電熱負荷 | I × 1.1〜1.25 | 周囲温度・電圧変動の誤差考慮 |
| 水銀灯・HID照明 | I × 1.4〜1.7 | 始動時の突入電流対策 |
| 電動機（じか入れ） | I × 3.0以下 | 始動電流(定格の5〜8倍)での不要遮断防止 |
