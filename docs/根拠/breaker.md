### 3.1 配線用遮断器の容量（3705-3表 右側）

参照ページ: **PDF p801**（紙面p719）

PDF p801の右側表は、**幹線の定格出力合計**（PDF p800左側の行）と、**電動機中最大のもの**（じか入れ始動・スターデルタ始動）の2軸で、用上最小の配線用遮断器容量を示すマトリクスである。

旧MDと現行 `MOTOR_BREAKER_TRUNK` のように「遮断器容量ごとに使用可能な最大kW範囲」へ単純化すると、行方向の「幹線の定格出力合計」が失われる。PDF p801を正とするには、下記のような2軸表として再構成する必要がある。

| 軸 | PDFでの意味 | 実装時の扱い |
|---|---|---|
| 行 | 幹線の電動機定格出力合計（PDF p800の 3, 4.5, 6.3 ... 86.2kW） | `kwTotal` として保持する必要あり |
| 列（上段） | じか入れ始動の電動機中最大のもの（0.75以下, 1.5, 2.2 ... 55kW） | `directLargestKw` 別に遮断器容量を保持 |
| 列（下段） | スターデルタ始動器使用の電動機中最大のもの（5.5, 7.5 ... 55kW） | `starDeltaLargestKw` 別に遮断器容量を保持 |

> 2026-06-01時点では、p801の全マトリクスをこのMDへ全文転記せず、構造上の要修正として扱う。現行 `src/app/breaker/constants.ts` の `MOTOR_BREAKER_TRUNK` はこの2軸構造を保持していないため、根拠表としては要再構成。

**備考**: 配線用遮断器の特性は3705節と資料3-7-5に従う。

## 5. 需要率（3605-11表）

参照ページ: **PDF p779**（紙面p697）

10kVA を超える容量に対する需要率：

| 建物の種類 | 需要率 |
|---|---|
| 住宅・寮・アパート・旅館・ホテル・病院・倉庫 | 50% |
| 学校・事務所・銀行 | 70% |

**備考**: 3605-11表の需要率は、電灯及び小形電気機械器具の容量合計が10kVAを超える場合、その超過容量に対して適用する。PDF p779の旅館の計算例では、電灯及び小形電気機械器具 30kVA、大形電気機械器具 5kVA、適用需要率50%として次の形で算出している。

```
最大使用負荷 = (30kVA - 10kVA) × 0.5 + 10kVA + 5kVA = 25kVA
```

したがって、初めの10kVAは100%で見込み、10kVA超過分に需要率を掛け、大形電気機器は別加算する。

## 7. 幹線の過電流遮断器選定（電技解釈 第148条）

### 7.1 法的上限の式

```
電動機負荷がある場合:
  I_B ≤ 3 × I_M + I_H
  (I_M > 50A の場合: I_B ≤ 2.75 × I_M + I_H)
  I_M: 電動機定格電流の合計
  I_H: その他の負荷電流の合計

電線の許容電流に基づく上限:
  I_B ≤ 2.5 × I_W
  I_W: 幹線の許容電流

上記2式のうち「小さい方」を上限とする
```

### 7.2 実務上の選定マージン

| 負荷種別 | 推奨選定係数 | 理由 |
|---|---|---|
| 一般電灯・電熱負荷 | I × 1.1〜1.25 | 周囲温度・電圧変動の誤差考慮 |
| 水銀灯・HID照明 | I × 1.4〜1.7 | 始動時の突入電流対策 |
| 電動機（じか入れ） | I × 3.0以下 | 始動電流(定格の5〜8倍)での不要遮断防止 |

## 9. PDF目視照合結果（2026-06-01）

PDF p567 / p779 / p796-p801を目視確認した結果、旧MDおよび現行 `src/app/breaker/constants.ts` には以下の差分が残っている。

| 対象 | 状態 | 確認結果 |
|---|---|---|
| `MOTOR_TABLE_SINGLE` | 要修正 | PDF p796-p797と、こう長・分岐回路遮断器・接地線に複数不一致あり。根拠MD本文はPDF値へ修正済み |
| `MOTOR_TABLE_TRUNK` | 要修正 | PDF p800と、電線管/VV・がいし引きのこう長に複数不一致あり。根拠MD本文はPDF値へ修正済み |
| `MOTOR_BREAKER_TRUNK` | 要再構成 | PDF p801は2軸マトリクス。現行実装の「遮断器容量→kWリスト」構造では根拠表を保持できない |
| `WELDER_TABLE` | 概ね一致 | PDF p567と主要値は一致。必要なら別途、全セル単位の再照合ログを残す |
| `DEMAND_FACTOR` | 数値一致 / 式説明修正 | PDF p779の需要率50%・70%は一致。旧MDの対象建物と計算式説明をPDFに合わせて修正 |

旧 `docs/根拠/照合レポート.md` には「修正済み」「完全一致」とする記述が残るが、今回のPDF目視では少なくとも電動機表の複数箇所で再修正が必要と判断する。
