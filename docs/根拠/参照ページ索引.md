# ELE-CAL 根拠参照ページ索引

> 作成日: 2026-06-01
> 目的: 6ツールについて、実装・根拠MD化・原本PDF目視の観点から「どのページを参照すべきか」を整理する。
> 前提: 原本PDFが一次根拠。MDは検索補助。実装値へ反映する前に、該当PDFページを目視確認する。

## 判定区分

| 区分 | 意味 |
|---|---|
| 必須 | 現行ツールの主要計算・表示に必要な根拠ページ |
| 条件付き | 現行では補助・未実装・設定次第だが、扱うなら必要なページ |
| 外部根拠 | 内線規程・公共建築だけでは不足し、JCS/JIS/メーカー資料などが必要な項目 |
| 範囲外 | 現行ツールでは参照しない。将来機能化する場合のみ扱う |
