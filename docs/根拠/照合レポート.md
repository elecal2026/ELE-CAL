### MOTOR_TABLE_SINGLE.motorBreakerDirect / motorBreakerStarDelta

constants.tsの値は内線規程の「電動機用超過目盛電流計の定格電流」とも、別の表とも一致しない。
**何の数値か出典不明。**

例: kw=0.75 で `motorBreakerDirect: 10` だが、PDF p797 では「電動機用超過目盛電流計の定格電流: 5A」。

---

## 原因の推測

constants.ts のヘッダーコメントには「**(内線規程 p.706-707 より)**」と書かれているが、

1. 紙面 p706 は実際には PDF の **p788 (3705節 配線設計の説明文)**
2. 3705-1表本体は **PDF p796**（紙面 p714）

→ **このコードを書いた時の参照ページが間違っている可能性が高い**。
→ おそらく旧版（JEAC 8001-2016 など）や別の参考書を引用した。
→ 第14版 (2022) で改定された数値と齟齬が出ている。

---

## 修正方針（提案）

### 方針A: 正本準拠で全面修正（推奨）

- `MOTOR_TABLE_SINGLE` の `breakerDirect` / `breakerStarDelta` を PDFp797 の値に書き換え
- `motorBreakerDirect` / `motorBreakerStarDelta` を「電動機用超過目盛電流計」と分かるよう **リネーム or 削除**
- ヘッダーコメントを `(内線規程JEAC8001-2022 3705-1表/PDF p796-797 より)` に更新

### 方針B: 現状維持 + 但し書き追加

- 既存ロジックの動作実績を維持
- README に「数値は2016版または独自解釈ベース」と明記
- 後日まとめて正本準拠版へ移行

### 方針C: 旧版から修正対象を絞る

- ブレーカー定格は配線用遮断器の規格上限なので、新版で小さくなった = 安全側へ寄った
- ELE-CAL が出してきた値は「マージンを多めに取った」結果と捉える
- ユーザー向け表示時に「内線規程準拠値より大きめに選定」と注記
