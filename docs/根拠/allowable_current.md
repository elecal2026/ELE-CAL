## 6. 算定上の重要原則

### 6.1 許容電流の補正計算式

```
I_corrected = I_base × K_temp × K_group
  I_base: 1340-1表 / 1340-2表 / 資料1-3-3 の基底許容電流
  K_temp: 周囲温度補正係数（1340-3表）
  K_group: 電流減少係数（1340-2表(その2)）
```

### 6.2 端数処理

電技解釈上の慣習として、**小数点以下第1位を7捨8入**して整数値とする。

### 6.3 単相3線式の電線数算入ルール

- 中性線・接地線・制御回路用電線は電線数に算入しない
- ただし「単相3線式2回路を同一管内に収める」場合は、中性線が2本あるため4本扱い

## 7. ELE-CAL現状値との対応関係

`src/app/allowable_current/page.tsx` の DATA オブジェクトとの対応：

| ELE-CAL内 | 内線規程の出典 |
|---|---|
| `IV.気中（単線）` | 1340-1表 単線 |
| `IV.気中（より線）` | 1340-1表 より線 |
| `IV.管内（3本以下）` | 1340-2表 同一管内電線数3以下 |
| `IV.管内（4本）` | 1340-2表 同一管内電線数4 |
| `CV.気中暗渠` | 資料1-3-3 気中暗きょ布設 |
| `CV.管路引入れ` | 資料1-3-3 管路引入れ布設 |
| `CV.直埋` | 資料1-3-3 直接埋設布設 |
| `CVD.*` | 資料1-3-3 2心CV |
| `CVT.*` | 資料1-3-3 3心CV(CVT) |
