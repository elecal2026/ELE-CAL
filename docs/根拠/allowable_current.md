## 4. 1340-3表 — 許容電流補正係数と周囲温度による減少係数計算式

参照ページ: p68（PDF p150）

PDF p150は、5℃刻みの数表ではなく、絶縁電線の種類ごとの補正係数と周囲温度による許容電流減少係数の計算式を示している。

| 絶縁電線の種類・施工条件 | 最高許容温度 (℃) | 許容電流補正係数 | 許容電流減少係数計算式 |
|---|---:|---:|---|
| IV電線（600V二種ビニル絶縁電線を除く）、RB電線（天然ゴム混合物に限る） | 60 | 1.00 | `R = sqrt((60 - theta) / 30)` |
| 600V二種ビニル絶縁電線、600Vポリエチレン絶縁電線（架橋ポリエチレン混合物を除く）、スチレンブタジエンゴム絶縁電線 | 75 | 1.22 | `R = sqrt((75 - theta) / 30)` |
| エチレンプロピレンゴム絶縁電線 | 80 | 1.29 | `R = sqrt((80 - theta) / 30)` |
| 600V架橋ポリエチレン絶縁電線 | 90 | 1.41 | `R = sqrt((90 - theta) / 30)` |
| 600Vけい素ゴム絶縁ガラス編組電線（接触による温度上昇等の影響を受けない施工） | 180 | 2.24 | `R = sqrt((180 - theta) / 30)` |
| 600Vけい素ゴム絶縁ガラス編組電線（その他） | 180 | 1.41 | `R = sqrt((90 - theta) / 30)` |
| 600Vふっ素樹脂絶縁電線（接触による温度上昇等の影響を受けない施工） | 200 | 2.14 | `R = 0.9 * sqrt((200 - theta) / 30)` |
| 600Vふっ素樹脂絶縁電線（その他） | 200 | 1.27 | `R = 0.9 * sqrt((90 - theta) / 30)` |

> 旧MDにあった「周囲温度5〜50℃の補正係数表」はPDF p150上では確認できないため削除した。必要な場合は、上記計算式または資料1-3-3の「基底温度による電流補正係数」ページを別途照合する。

## 5B. HIV（600V二種ビニル絶縁電線）の許容電流（2026-06-07 PDF照合）

参照ページ: p150（1340-3表）、p151（計算例）

HIVは内線規程に独立の許容電流表を持たない。**1340-3表の許容電流補正係数 1.22 を、IV電線の基準許容電流に乗じて求める**のが規程の方式。

PDF p151の計算例（原文）:
> 1340-1表より2.0mmの許容電流は35A。1340-3表より600V二種ビニル絶縁電線の許容電流補正係数は1.22であり、35×1.22=42.7≒42A（小数点以下1位を7捨8入すると42Aとなる）

→ **HIV許容電流 = IV基準許容電流 × 1.22（小数点以下第1位を7捨8入）**

- 上記計算例は **1340-1表（気中・がいし引き）基準** に対するもの。気中HIVはこの方式で確定。
- **管内（電線管3本以下等）への 1.22 適用も規程準拠で確定**（2026-06-07 別エージェント検証）。1340-3表の表頭は「絶縁電線の許容電流補正係数」で、1.22は布設条件でなく絶縁物温度75℃で決まる汎用係数。p151計算例(2)は同じHIVを金属管配線で使う場合に「1.22 × 周囲温度減少係数(0.91)」と独立に掛ける構造を示しており、管内基準値（1340-2表＝既に電流減少係数込み）に 1.22 を乗じる扱いが正しい。
  → **HIV管内（3本以下） = IV管内（3本以下） × 1.22（7捨8入）** で実装反映済み。

## 6. 算定上の重要原則

### 6.1 許容電流の補正計算式

```
I_corrected = I_base × K_temp × K_group
  I_base: 1340-1表 / 1340-2表 / 資料1-3-3 の基底許容電流
  K_temp: 周囲温度による補正・減少係数（1340-3表、資料1-3-3の補正係数表）
  K_group: 電流減少係数（1340-2表(その2)）
```

### 6.2 端数処理

電技解釈上の慣習として、**小数点以下第1位を7捨8入**して整数値とする。

### 6.3 単相3線式の電線数算入ルール

- 中性線・接地線・制御回路用電線は電線数に算入しない
- ただし「単相3線式2回路を同一管内に収める」場合は、中性線が2本あるため4本扱い
