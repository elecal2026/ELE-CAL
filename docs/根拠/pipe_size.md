## 1. 占有率ルール（32% / 48%）

### 1.1 基本ルール（32%）— 3110-5（金属管）/ 3115-4（合成樹脂管）

参照ページ: PDF p355（紙面p273）、PDF p368（紙面p286）

異なる太さの絶縁電線を同一管内に収める場合の管の太さは、
**電線の被覆絶縁物を含む断面積の総和が管の内断面積の32%以下** となるように選定する（勧告）。

### 1.2 緩和条件（48%）— 3110-6（金属管）/ 3115-5（合成樹脂管）

参照ページ: PDF p354（紙面p272）、PDF p369（紙面p287）

**管の屈曲が少なく、容易に電線の引き入れ及び引き替えができる場合に限り**、
電線の被覆絶縁物を含む断面積の総和が管の内断面積の **48%以下** まで許容する。

### 1.3 ケーブル特例

ケーブル（CV等）を保護管に通す場合、管の内径はケーブル外径の **1.5倍以上** が推奨。

## 6. 計算アルゴリズム

### 6.1 電線種類による自動判定

- `IV` / `HIV` を選択した場合は、絶縁電線の占積計算を使用する。
- `VVF` / `VVR` / `CV` / `CVD` / `CVT` を選択した場合は、ケーブル防護管選定を使用する。
- 利用者に計算方法の切替操作は求めず、選択した電線種類から自動判定する。
- 共通電線マスターの仕様はすべて選択可能とし、計算データ又は仕上り外径データがない仕様は結果を `該当なし` とする。

### 6.2 絶縁電線の占積計算

```
1. 電線を全てリストアップ（線種・太さ・本数）
2. 各電線の断面積を 3110-7表 から取得
3. 各電線の補正係数を 3110-8表 から取得（金属管のみ）
4. 補正後断面積を合計
   合計断面積 = Σ(電線断面積 × 補正係数 × 本数)
5. 占有率モードを決定（32% or 48%）
   - 屈曲少・引き替え可能 → 48%
   - それ以外 → 32%
6. 該当配管種類の内断面積表から、合計断面積以上の最小管サイズを選定
```

### 6.3 ケーブル防護管選定

```
1. ケーブル種類・仕様を選択
2. 根拠確認済みの仕上り外径を取得
3. 必要内径 = ケーブル仕上り外径 × 1.5
4. 各配管種類から、必要内径以上となる最小サイズを選定
```

- 1本のケーブルを1本の防護管に収める場合を対象とする。
- 複数ケーブルを同一防護管へ収める計算式は、本ツールでは扱わない。
- `FEP` はケーブル防護管選定のみで表示する。

## 7. ELE-CAL実装との対応

`src/app/pipe_size/page.tsx` と共通データとの対応：

| ELE-CAL内 | 内線規程の出典 |
|---|---|
| 共通電線マスター | ラクダ掲載範囲の選択肢と統一表記 |
| `IV_AREA_BY_SPEC_ID` | 3110-7表（IV電線の被覆込み断面積） |
| `PIPE_DEFINITIONS` 金属管3種 | 3110-9〜11表 |
| `PIPE_DEFINITIONS` VE | 3115-9表 |
| `PIPE_DEFINITIONS` PF管・CD管 | 3115-10表 |
| `PIPE_DEFINITIONS` FEP | FEPは内線規程本文外（JIS C 3653）。ケーブル防護管選定のみ |
| `src/data/cable-physical.ts` | ラック選定と共用するケーブル仕上り外径の代表値 |
| `IV_CORR20` (補正係数2.0) | 3110-8表 単線1.6/2.0 |
| `IV_CORR12` (補正係数1.2) | 3110-8表 単線2.6/3.2、より線5.5/8 |

実装確認（2026-06-14）:

- 共通電線マスターから、電線種類と電線仕様を選択する。
- `IV/HIV` とケーブル類を選択した時点で計算方法を自動判定する。
- 絶縁電線計算へケーブルの断面積を混在させない。
- `HIV`、IVの表外サイズ、仕上り外径データのないケーブル仕様は選択可能だが、結果を `該当なし` とする。
- 心数表記は共通マスターの `単心` / `2心` / `3心` / `4心` を使用する。
- ケーブル外径データは `cable_rack` と共通化した。
- `npm run build` とブラウザ表示を確認済み。

## 8. 実務上の注意

### 8.1 屈曲箇所数による実態

1ルート内のノーマル（曲がり）が3箇所以上ある場合、入線時の摩擦抵抗が激増する。
このため、計算上32%に収まっていても、配管長が長く曲がりが多いルートでは **計算より1サイズ上の管** を採用する。

### 8.2 アース線の取り扱い

幹線計算の主導体に含まれないアース線（IV電線等）を同じ管に入れる場合、
その分の占有率を **忘れずに合算** すること。

### 8.3 入線本数の最大規定（3115-6表）— 同太さの場合

参照ページ: PDF p369（紙面p287）

異なる太さでなく、同太さの絶縁電線を同一管内に収める場合、
**電線本数 10本まで** の規定がPF管・CD管にあり（3115-6表）。

### 8.4 屈曲少・引き替え容易での最大電線本数（3115-7表）
