# 集合住宅幹線設計 — 内線規程からの根拠資料

> 出典: 内線規程 第14版 JEAC 8001-2022（電気技術規程需要設備編）
> 担当ツール: `/apartment_main`（実装中 2026-05-27〜）
> このMDは集合住宅向け幹線設計ツールを実装する際の **根拠データ集（数値の一次ソース）**。
> **実装仕様・方針は [`docs/apartment_main_実装指示.md`](../apartment_main_実装指示.md) を参照すること**。本資料は内線規程の抜粋に徹する。

## 0. 全体フロー（集合住宅 専用）

```
[戸数] [住戸面積] [全電化/一般] [電気温水器有無]
       ↓
[戸当たり設備負荷容量]を 3-6-1表 or 3-6-2表 から決定
       ↓
[幹線の総合需要率] を「幹線の需要率表」から決定（戸数依存）
       ↓
[想定最大負荷 P_max] = 戸当たり負荷 × 戸数 × 需要率
       ↓
[単相3線式 100/200V] [三相3線式 100/200V] 別に電流算出
       ↓
[CVTケーブル最小太さ]・[配線用遮断器の定格電流]を表から決定
```

## 1. 戸当たり設備負荷の算定（3605-1）

参照ページ: p681-682

### 1.1 標準負荷（3605-1表）

| 建物種類 | 標準負荷 (VA/m²) |
|---|---|
| 工場・公会堂・寺院・教会・劇場・映画館・寄席・ダンスホール・倉庫・農家の納屋など | 10 |
| 学校・銀行・旅館・ホテル・クラブ・病院・学校・料理店・喫茶店・飲食店・公衆浴場 | 20 |
| 事務所・銀行・百貨店・商店・理髪店・美容院 | 30 |
| **住宅・アパート** | **40** |

### 1.2 部分的標準負荷（3605-2表）

| 部分 | 標準負荷 (VA/m²) |
|---|---|
| 廊下・階段・手洗所・倉庫・貯蔵室 | 5 |
| 講堂・観覧室 | 10 |

### 1.3 標準負荷外の加算VA数

下記は標準負荷とは別に加算する：

- **住宅・アパート（1世帯につき）**: 500〜1,000VA
- 商店のショウウィンドー: 開口長1mにつき **300VA**
- 屋外の広告灯・電気サイン・ネオンサインなど: 個別VA数
- 劇場・映画館・ダンスホールなどの舞台照明・映画撮影用などの特殊な電灯負荷: 個別VA数

### 1.4 設備負荷容量 計算式

```
設備負荷容量 P = P_A + Q + B + C
  P_A: 建物の床面積 × 3605-1表の標準負荷 (Q部分を除く)
  Q  : 3605-2表で別計算する部分の床面積 × 3605-2表の標準負荷
  B  : 3605-1表の建物種類によらず加算する標準負荷部分のVA数
  C  : 加算するVA数（住宅500〜1,000VA、店舗ショウウィンドー300VA/m など）
```

### 3.1 計算前提

- 住戸面積 **100m²** 当たり、戸当たり設備負荷容量 **4.4kVA/戸** （マイコン制御型または23時一斉始動型の電気温水器を含む）
- マイコン制御型電気温水器の場合と、23時一斉始動型電気温水器の場合で需要率が異なる

### 3.2 想定最大負荷の算出式

```
P_θ = (P × d × k) + n × P_W (kVA)
  P: 戸当たり設備負荷容量 4.4kVA/戸（100m²の場合）
  d: 一般電力の需要率 (%)
  k: 重畳率
  P_W: 電気温水器の想定負荷
  n: 戸数

ただし、P: 戸当たり設定負荷 = 0.06 (kVA/m²) × S (m²) + 4 (kVA/戸)
  S: 住戸面積
```

## 4. 一般住宅の裕度ある配線設計（資料 3-6-5）

参照ページ: p933

### 4.1 想定範囲

集合住宅・全電化集合住宅は対象外。一般住宅（戸建て）における裕度ある設計の参考例。

### 4.2 最大需要電力の想定（住宅面積別）

| 住宅面積 (m²) | 最大需要電力 (kVA) |
|---|---|
| 50 (15坪) 以下 | 4 |
| 70 (20坪) | 5 |
| 100 (30坪) | 6 |
| 130 (40坪) | 7 |
| 170 (50坪) | 8 |

### 4.3 算出式

```
P_m = 40 (VA/m²) × 住宅面積 (m²) × 1,000 〜 2,500 (VA)
```

最大需要電力の想定が 1,000〜2,500VA は下表を目安とする：

| 住宅面積 | 加算値 (VA) |
|---|---|
| 50以下 | 2,500 |
| 50超〜100 | 2,000 |
| 100〜150 | 1,500 |
| 150以上 | 1,000 |

**備考**:
1. 上記の最大需要電力には、照明やエアコン(1台)が含まれているが、2台目以降のエアコンや大容量機器は別途加算する必要がある

## 6. 分岐回路数の決定（3605-3）

参照ページ: p685-686

### 6.1 決定ルール

```
1. 設備負荷容量を 3605-1（負荷の想定）により算出
2. 想定された設備負荷容量を、回路の使用電圧 100V の 15A・20A（負荷の見方）
   で割って必要最小分岐回路数を決定
3. 1,500VA を超えるものは、原則として別途専用回路を設ける
4. 大形電気機械器具（10Aを超える）には、別途専用回路を設ける
5. 連続負荷の場合は分岐過電流遮断器の定格電流の80%を超えないこと
```

### 6.2 計算例（3605-3図 p686）

```
住宅部分: 120m² × 40VA/m² = 4,800VA
店舗部分: 50m² × 30VA/m² = 1,500VA
廊下部分: 10m² × 5VA/m² = 50VA
住宅加算VA: 1,000VA
ショウウィンドー: 関口長3m × 300VA = 900VA

合計: P + Q + B + C = 4,800 + 1,500 + 50 + 1,000 + 900 = 8,250VA

回路数: 8,250 ÷ 1,500 = 5.5 → 6回路
+ 大形電気機械器具(11Aルームエアコン): 1回路
→ 合計 7 回路
```

## 7. 実装方針

> **本資料は内線規程の抜粋（根拠データ）に徹する。実装仕様は別ドキュメントを参照すること。**
>
> 実装指示書: [`docs/apartment_main_実装指示.md`](../apartment_main_実装指示.md)
>
> 実装指示書には「内線規程準拠・独自補正なし・住戸面積100m²固定・一般40戸/全電化16戸まで」の方針が明記されている。実装者は実装指示書を最上位の根拠とし、本資料は数値の一次データとして参照する位置づけ。

### 7.1 本資料の利用方法

- §1〜§6 の数値表は、実装側の `data.ts` に1行1行転記する際の **唯一の真実** として扱う
- 転記後の検算チェックは、表の値どうしの整合（例: 想定最大負荷と電流値）を確認する用途に使う

### 7.2 内線規程上の前提（実装時に意識すべき注記）

- **基底温度**: CVTケーブル最小太さの表は基底温度40℃の値（≠ 一般的な30℃）
- **3心扱い**: 単相3線式は3心扱いで計算
- **マイコン制御 vs 23時一斉**: マイコン制御の方が需要率が低く、電線も細くて済む傾向

### 7.3 本資料に含まれないが内線規程上は存在しうる範囲

以下は本資料の表に含まれていないが、内線規程の他章 / 追加撮影 / 補間で扱える可能性がある:

- 全電化集合住宅 17〜25戸の配線用遮断器・CVTケーブル（表で「—」となっている部分）
- 全電化集合住宅 26戸以上（p924上半分が切れているため要再撮影）
- 住戸面積 100m² 以外への補正（資料 §3.2 に式があるが、表との整合は本資料の範囲では完結していない）

これらは本ツールの **対象外**（実装指示書 §10 参照）。将来の拡張要望時に再検討する。
