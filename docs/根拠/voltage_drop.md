## 1. 電圧降下の許容限度（1310-1）

参照ページ: **PDF p120**（紙面p38）

### 1.1 基本ルール

低圧配線中の電圧降下は、幹線及び分岐回路にて、それぞれ標準電圧の **2%以下** とすること（勧告）。
ただし、電気使用場所内の変圧器により供給される場合の幹線の電圧降下は、**3%以下** とすることができる。

### 1.2 こう長が60mを超える場合の許容限度 — 1310-1表

| 供給変圧器の二次端子又は引込線取付点から最遠端の負荷に至るまでの電線のこう長 (m) | 電気使用場所内に設けた変圧器から供給を受ける場合 | 一般送配電事業者から低圧で供給を受ける場合 |
|---|---|---|
| 120 以下 | 5%以下 | 4%以下 |
| 200 以下 | 6%以下 | 5%以下 |
| 200 超過 | 7%以下 | 6%以下 |

### 1.3 注記

- 引込線取付点から引込口までの部分も特別注意（注1）
- 電気使用場所内に設けた変圧器から供給を受ける場合は、主配電盤までの部分も特別注意（注2）
- 配線方式、負荷状況などにより電圧降下値については、資料1-3-2を参照（注3）
- 系統連系型小出力太陽光発電設備等の連携時は、引込線取付点からの電圧降下は3594節を参照（注4）

## 2. 詳細計算式（インピーダンス法）

参照ページ: **PDF p875**（紙面p793）資料1-3-2

### 2.1 集合住宅幹線・長距離・大電流の場合

```
e = K₁ × I × (R·cosθ + X·sinθ) × L
  e: 電圧降下 (V)
  K₁: 配線方式による係数（下表）
  I: 通電電流 (A)
  R: 電線1km当たりの交流導体抵抗 (Ω/km)
  X: 電線1km当たりのリアクタンス (Ω/km)
  cosθ: 負荷力率
  L: 電線のこう長 (km)
```

### 2.2 配線方式別係数 K₁

| 配線方式 | K₁ | 対象電圧 |
|---|---|---|
| 単相2線式 | 2 | 線間 |
| 単相3線式 | 1 | 線間 |
| 三相3線式 | √3 | 線間 |

参照: 資料1-3-2「4. 600V架橋ポリエチレン絶縁ビニル外装ケーブルのインピーダンス表」及び「5. 600Vビニル絶縁ビニル外装ケーブルのインピーダンス表」

## 3. 簡易計算式

参照ページ: **PDF p875**（紙面p793）資料1-3-2

### 3.1 適用条件

- 屋内配線などで比較的に電線こう長が短く、また電線が細い場合
- 表皮効果や近接効果などによる導体抵抗値とリアクタンス分を無視してもらえない場合

### 3.2 計算式

| 配線方式 | 電圧降下式 | 対象電圧 |
|---|---|---|
| 単相2線式 | `e = (35.6 × L × I) / (1000 × A)` | 線間 |
| 三相3線式 | `e = (30.8 × L × I) / (1000 × A)` | 線間 |
| 単相3線式・三相4線式 | `e = (17.8 × L × I) / (1000 × A)` | 対地 |

```
e: 電圧降下 (V)
I: 負荷電流 (A)
L: 電線のこう長 (m)
A: 使用電線の断面積 (mm²)
```

> ℹ️ ELE-CAL の `COEFF = { '1phase3wire': 35.6, '3phase3wire': 30.8 }` はこの式が出典。
> ⚠️ 単相3線式は本来 17.8（対地）だが、ELE-CALは35.6（単相2線式と同じ）にしている。要確認。

## 5. 実務上の重要事項

### 5.1 簡易式の限界

簡易式は、長距離・大電流の幹線では値を **過小評価する傾向** がある。
集合住宅幹線や大規模工場の幹線では、インピーダンス法による詳細計算が推奨される。

### 5.2 2%基準の意義

- **機器の安定稼働**: PC・精密測定機器・インバーター機器は電圧変動に敏感
- **熱損失（銅損）の低減**: 電気代の無駄を減らし省エネ
- **将来の負荷増設マージン**: 末端電圧の品質を確保

### 5.3 三相3線式の係数

```
K = √3 ≈ 1.732
30.8 ≈ √3 × 17.8
35.6 ≈ 2 × 17.8
```

つまり「17.8」は対地電圧降下（中性線基準）の係数で、配線方式によって倍率が異なる。

## 6. ELE-CAL現状コードとの対応

`src/app/voltage_drop_v2/page.tsx` との対応：

| ELE-CAL内 | 内線規程の出典 |
|---|---|
| `COEFF['1phase3wire'] = 35.6` | **要確認** — 単相3線式は本来17.8（対地）。35.6は単相2線式の係数 |
| `COEFF['3phase3wire'] = 30.8` | 簡易式 三相3線式 |
| `STD.transformer` (専用変圧器の許容限度) | 1310-1表「電気使用場所内に設けた変圧器」 |
| `STD.utility` (一般送配電事業者の許容限度) | 1310-1表「一般送配電事業者から低圧」 |
| 60m以下の幹線2%/分岐2%（合計4% or 5%） | 1310-1 本文 |

### 6.1 STD構造との照合

ELE-CAL の `STD` テーブル：

```javascript
transformer: [
  { min: 0, max: 60, branch: 2, trunk: 3, total: null },
  { min: 60, max: 120, total: 5 },
  { min: 120, max: 200, total: 6 },
  { min: 200, max: Infinity, total: 7 },
]
utility: [
  { min: 0, max: 60, branch: 2, trunk: 2, total: null },
  { min: 60, max: 120, total: 4 },
  { min: 120, max: 200, total: 5 },
  { min: 200, max: Infinity, total: 6 },
]
```

→ **完全に1310-1表と一致**（trunk:3, branch:2 → 専用変圧器 / trunk:2, branch:2 → 一般送配電）
